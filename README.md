## 关于inc

这是一个基于SpringMVC+Springboot2+Mybatisplus+Vue+ElementUI+Axios的前后端分离工程。  
本工程是一个具有权限管理、菜单管理、文件上传/下载的图书管理系统，不同的用户登入可操作或查询权限内的数据信息。
<br>
<br>
前端工程：inc-view  
后端工程：inc-admin
